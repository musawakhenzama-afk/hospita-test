# BRIGHTLEARN DATA ANALYTICS
Case Study : Hospital Readmission Analysis
Task: Analyze the patient readmission dataset and present meaningful insights to the Hospital Head Doctor.
Instructions:
•
You may use any tools you prefer for analysis and visualization.
•
Each student must perform their own unique analysis , explore the data from your own perspective.
•
Identify trends, patterns or factors behind patient readmissions.
•
Provide data-driven recommendations to improve hospital performance and patient care.
•
Submit your work via GitHub.
Goal: Show your ability to independently explore data, draw insights, and communicate findings clearly and professionally.
